Methods

createXMLFile()

createXMLFile(  $links) : string

Uses the $links data gathered from the links database table to loop through and generate XML in sitemap format, as specified in the https://www.sitemaps.org/protocol.html#sitemapXMLExample webpage As well as the URL, the createXMLFile method uses the page type to set the Priority, based on a city page requiring 0.7, an activity (or event) requiring a 0.5 priority and any other page requiring a 1.0 priority. Last Modified Date information was not available to be used.

Used https://programmerblog.net/how-to-generate-xml-files-using-php/ to help build method

Parameters

$links

Array of links from db

Returns

string —

$xmloutput - contains XML for HTML rendering
